Train Pipeline

标注与训练闭环

训练模块优先服务现有业务模型沉淀：导入数据、预标注、人工修正、训练 YOLO26、评估后发布到当前检测链路。

01 数据集
支持 ZIP 图片、视频抽帧、历史结果图一键回流。
02 预标注
优先用 `yolov8s-worldv2.pt` 做 prompt 预标注，再人工修正。
03 训练与发布
用 `yolo26n.pt` / `yolo26s.pt` 训练闭集模型，发布回 `model/`。

训练预设

Quick
`yolo26n.pt` / `imgsz=640` / `epochs=30`
Standard
`yolo26s.pt` / `imgsz=640` / `epochs=80`
Low Memory
`yolo26n.pt` / `imgsz=512` / small batch
发布后把 `best.pt` 复制到 `model/`，当前上传检测页会自动把新模型加入下拉列表。
