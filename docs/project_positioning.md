# 🎯 项目定位文档（道）

> 版本：v1.0 | 日期：2026-04-18

---

## 一句话价值主张

> **「让基层民警无需 AI 专业知识，即可完成从图片采集、智能识别到线索研判、任务派发的全链路闭环。」**

---

## 项目名称

**多骑手智慧研判平台**（Multi-Rider）

---

## 目标用户

| 用户角色 | 具体是谁 | 使用场景 |
|---------|---------|---------|
| **一线办案民警** | 交警、治安民警、专班民警 | 从卡口照片中筛查违法二轮车、翘车头飙车行为 |
| **人脸布控民警** | 刑侦、治安 | 批量比对人脸照片，快速锁定嫌疑人身份 |
| **专班指挥人员** | 专班负责人 | 下发任务、查看研判结果、发送短信通知 |
| **技术维护人员** | 你自己（目前） | 模型训练、系统维护、数据管理 |

**核心用户画像**：30-45岁基层民警，不懂 AI 技术，使用内网 Chrome 88/109 浏览器，希望系统"打开就能用，点一下就出结果"。

---

## 解决什么核心痛点

### 痛点1：海量卡口图片人工筛查效率极低

```
现状：「飙车炸街」专班需要从数万张卡口照片中人工辨别翘车头、
      违法改装等行为，一个民警看一天也只能筛几百张。

本平台：AI 自动检测 + 批量处理，数万张照片 10-30 分钟出结果。
```

### 痛点2：人脸比对依赖上级系统排队等候

```
现状：基层要做人脸比对需要提交到上级系统，排队等审批，
      响应时间长、无法自主掌握节奏。

本平台：本地部署人脸识别引擎，基层自主完成人脸比对，秒级响应。
```

### 痛点3：模型训练门槛高，基层无法自主迭代

```
现状：新出现一种违法行为（如某种改装车型），想让 AI 识别它，
      必须找专业技术人员训练模型，周期长、沟通成本高。

本平台：提供可视化的数据标注和一键训练功能，
       基层民警自己标注几十张图，系统自动训练出可用模型。
```

### 痛点4：研判结果和任务派发割裂

```
现状：AI 识别出结果后，还要人工录入到别的系统去派发任务、
      发短信通知，流程断裂，容易遗漏。

本平台：检测→识别→研判→派发→短信通知，全链路打通，一气呵成。
```

## 与竞品的差异化

### 竞品对比

| 对比维度 | 商业 AI 平台 | 上级统建系统 | **本平台 (Multi-Rider)** |
|---------|-------------|-------------|------------------------|
| **部署位置** | 云端/厂商机房 | 省厅/市局机房 | **基层内网本地部署** |
| **数据安全** | 数据出内网 ⚠️ | 数据上传省厅 | **数据不出本地 ✅** |
| **响应速度** | 需要排队 | 需要审批 | **秒级响应** |
| **采购成本** | 几十万到百万 | 免费但不可控 | **零采购成本（自研）** |
| **定制能力** | 需要厂商配合 | 无法定制 | **自主可控，随时调整** |
| **模型迭代** | 依赖厂商 | 依赖上级 | **基层自主训练迭代** |
| **全链路闭环** | 通常只做识别 | 通常只做比对 | **检测→识别→训练→派发全打通** |

### 三大核心差异

```
差异1：「全链路闭环」
       别人做一个环节，我们打通整个链条。
       从数据采集 → AI 检测 → 人脸识别 → 模型训练 → 任务派发 → 短信通知，
       一个平台全搞定。

差异2：「基层自主可控」
       不依赖云端、不依赖厂商、不依赖上级审批。
       数据不出内网，模型自己训练，节奏自己把握。

差异3：「实战验证」
       不是实验室产品，已在「飙车炸街」专班实际使用，
       有真实案例数据和效果对比。
```

## 技术定位

```
定位：基层公安AI实战应用平台
类型：内网部署的 Web 应用（B/S 架构）
架构：Flask Web 进程 + Worker 后台进程 + SQLite 任务队列
数据源：KingbaseV8（人脸底库）+ Oracle 11g（卡口照片 / 短信平台）
AI引擎：YOLO 系列（目标检测）+ InsightFace（人脸识别）
部署：Ubuntu 22.04 LTS 虚拟机（16核16G/500G）
维护：单人维护，追求简单可靠
```

## 一句话总结

> **Multi-Rider 是一个部署在基层内网的 AI 实战研判平台，
> 让不懂 AI 的民警也能完成从图片智能识别到任务派发的全流程，
> 核心优势是「全链路闭环 + 基层自主可控 + 实战验证」。**
