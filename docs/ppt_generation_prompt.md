## 三、技术与能力（核心说服点）

1. **多模态视像侦查**：YOLO-World 开放词表 + 飙车炸街私有训练模型做视频抓拍批量检测，
   InsightFace 人像比对以图搜人；**普通办公电脑 CPU 即可推理，无需 GPU**；
   民警可自主完成 采样→标注→训练→注册 全流程，模型自主可控。
2. **五维量化风险评分**：**案件 30 + 行为 25 + 家庭 20 + 教育 15 + 社交 10 = 100 分**，
   映射 **极高危/高危/中危/低危/正常 五级**，结果与明细持久化、留历史、可批量重算。
3. **Neo4j 关系图谱团伙发现**：构建"人—案—学校—监护人"异构关系图，按风险等级着色；
   **Louvain 社区发现 + 介数中心性 + 两人最短关系路径**，
   实现"发现一人、锁定一团、预警一片"。
4. **公安大模型 + RAG 智能研判（核心创新，紧扣 AI+ 方向）**：对接公安内网"锐智"大模型，
   叠加 **RAG 知识库 + 向量检索（Embedding）+ 重排（Rerank）**，提供
   **个人犯罪风险研判 / 侵财案件自动串并 / 办案法律问答** 三类能力，秒级输出结构化报告；
   大模型不可用时自动**降级为纯规则/纯检索**（有应急预案）。
5. **一人一档多源融合**：跨公安、教育、民政按身份证号归集，自动补全监护、就学
   （在校/辍学/失学/逃学）、行为、住宿、关联等维度。
6. **国产化与自主可控/信创**：全栈**国产 KingBase V8 数据库** + 开源组件
   （Python 3.12+Flask、Neo4j、SQLite 任务队列+后台 Worker、Jinja2+Tailwind 离线构建）；
   **纯公安内网离线部署，数据不出公安网**；零商业采购、零外包。

**核心闭环流程**：视像筛查 → 画像评分 → 图谱串并 → 闭环处置
（高危线索一键推送综治/指挥平台，短信+警务终端提醒，处置回执回流、样本反哺模型迭代）。
