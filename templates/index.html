Guide

使用说明

📦 ZIP 图片包
将 JPG / PNG / BMP / TIFF 图片打包成 ZIP 上传，推理后只保留触发检测的图片，重新打包下载。
🎬 视频文件
支持 MP4 / AVI / MOV / MKV。按帧采样间隔每隔 N 帧取一帧推理，命中的帧保存为 JPG 打包下载。
⚠️ 注意
单文件最大 500 MB；服务重启后进行中的任务会丢失，请勿上传过大的文件。
训练流程

数据集管理

这是训练模块的第一个真实入口。现在可以先创建本地数据集；后续再逐步补上 ZIP 导入、 历史结果回流、图片标注和训练任务。

创建后会自动生成本地数据集目录，并初始化 images/、 labels/、 splits/ 和 exports/。
ZIP 导入

批量导入图片

选择已有数据集后，上传一个包含图片的 ZIP 包。系统会自动解压有效图片到 images/ 目录，并记录图片元数据用于后续标注和训练。

支持 ZIP 内包含 jpg / jpeg / png / bmp / tif / tiff / webp 图片。非图片文件会自动跳过，导入结果会同步更新到下方数据集列表。
第一阶段只做图片归档和元数据入库。后续会在这批图片上继续接历史结果回流、标注和训练任务。
